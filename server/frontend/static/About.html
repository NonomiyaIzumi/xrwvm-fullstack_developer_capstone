About Us

Cars Dealership is a national car retailer with dealer branches across the United States. This site was built as the Full Stack Application Development Capstone project: a Django backend embedding a React single-page app, a Node.js/MongoDB microservice for dealers and reviews, and a Flask microservice for review sentiment analysis.
